This is free version of loot analyzer. This is community project, so every improvement can be done by anyone.

**IF YOU DONT WANT USE EXE FILE then:**

1. Press Green CODE and download as ZIP.
2. Install python 3 from https://www.python.org/downloads/
3. Open your Command Prompt (CMD) and run: pip install pygame
4. Place .bat and .py into Jinteia folder.
5. Run the bat.
6. Use ingame the white box for yangs and loot to have updates in files.

**IF YOU WANT TO USE EXE FILE then:**

1. Download Jinteia-Portable.exe on Releases.
2. Place .exe file into Jinteia folder.
3. Run the .exe
4. Use ingame the white box for yangs and loot to have updates in files.
